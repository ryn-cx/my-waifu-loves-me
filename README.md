# My Waifu Loves Me

Anime and manga recommendation explorer that visualizes relationships between media as an interactive graph. Search for anime or manga, load their recommendations from [AniList](https://anilist.co), and explore connections visually using a force-directed graph.

## Features

- Search for anime and manga via the AniList GraphQL API
- Interactive graph visualization of media recommendations using Sigma.js and graphology
- Import your AniList user list to overlay watch/read status on the graph
- Filter recommendations by status, popularity, year range, and minimum connections
- Dark mode support (but it's ugly)
